# 前言
这是我的德语学习笔记，希望能比较系统地概括德语的基础知识。
由于德语和英语的亲缘关系很近，我会多比较德语和英语的异同点。



# 字母表
德语字母和英语字母相似，只是多了4个字母：ä、ö、ü和ß（注意最后一个字母不是希腊字母β）。如果计算机环境打不出来，也可以用ae、oe、ue和ss代替。

# 发音

## 元音

德语的元音字母有8个，和英语一样，每个元音字母的发音有长短之分。下面用国际音标表示。

| 字母 | 长音 | 短音 |
| ---- | ---- | ---- |
| a    | /aː/ | /a/  |
| ä    | /ɛː/ | /ɛ/  |
| e    | /eː/ | /ɛ/  |
| i    | /iː/ | /ɪ/  |
| o    | /oː/ | /ɔ/  |
| ö    | /øː/ | /œ/  |
| u    | /uː/ | /ʊ/  |
| ü    | /yː/ | /ʏ/  |

总的来说，长音和短音主要是时长的区别，但是也伴随着嘴巴开闭程度的变化。对于除了e以外的字母来说，
嘴巴开闭程度的变化带来的区别不是很明显，可以忽略，但是对于e来说，长音和短音听感上会有很大差别。

元音字母的长短音拼读规则是
- 双写元音字母或者元音字母后面有h，则发长音，例如：aa、eh。
- 如果是单独的元音字母，后面没有或只有一个辅音字母的，发长音，后面有多于一个辅音字母的，发短音。
- 特殊规则：ie相当于长音的i。

下面粗略地描述这些字母的发音。这样发音保证可以听懂。
| 字母 | 发音                                                     |
| ---- | -------------------------------------------------------- |
| a    | 类似中文的“啊”                                           |
| ä    | 类似英文中bet的e                                         |
| e    | 短音同ä。长音介于i和ä之间（嘴巴开闭程度），听起来有点像i |
| i    | 类似中文的“一”                                           |
| o    | 类似中文的“哦”                                           |
| ö    | 和ä动作一样，只是要圆唇                                  |
| u    | 类似中文的“乌”                                           |
| ü    | 类似中文的“于”，具体来说，动作和i一样，但是要圆唇        |

## 双元音

| 字母组合    | 发音 |
| ----------- | ---- |
| ei/ai/ey/ay | /aɪ/ |
| au          | /aʊ/ |
| eu/äu       | /ɔʏ/ |

# 词性

德语的词性和英语完全相同，不赘述。

# 名词的性

和欧洲大多数语言一样，德语的名词可分为不同的性。单数名词可分为阳性、阴性和中性，复数也可看作一种“性”。

# 名词的格

英语中，格基本上退化了，只剩下部分代词的主格和宾格的区分。德语则有4种格，一般翻译成第一格、第二格、第三格和第四格，
也有翻译成主格、属格（或所有格、所属格）、与格和宾格的。我觉得和数字相比，人脑还是更擅长记忆具体的东西，所以我使用后者。
这些格的用处见下表。

| 格   | 用处                       |
| ---- | -------------------------- |
| 主格 | 主语、表语                 |
| 属格 | 定语、少数介词短语         |
| 与格 | 间接宾语、部分介词短语     |
| 宾格 | （直接）宾语、部分介词短语 |

下面的英语例子可以帮你快速理解上面的内容（除了介词短语部分）。

This(主) is my friend's(属) book(主). My friend(主) gave me(与) the book(宾).

# 复数名词的变形

英语名词的复数一般是加-s或者-es，不规则变化极少。不幸的是，德语的复数名词是没有变化规则的，只能死记硬背。

# 冠词

和英语相比，德语不仅有定冠词、不定冠词，还额外多了一种否定冠词。德语的定冠词和不定冠词用法和英语基本上是一样的。

## 定冠词

英语里面定冠词只需要一个the就搞定了，而德语里面定冠词需要根据名词的性和格来变化。

|      | 阳性 | 中性 | 阴性 | 复数 |
| ---- | ---- | ---- | ---- | ---- |
| 主格 | der  | das  | die  | die  |
| 宾格 | den  | das  | die  | die  |
| 与格 | dem  | dem  | der  | den  |
| 属格 | des  | des  | der  | der  |

## 不定冠词

下面是德语中不定冠词的变化。

|      | 阳性  | 中性  | 阴性  |
| ---- | ----- | ----- | ----- |
| 主格 | ein   | ein   | eine  |
| 宾格 | einen | ein   | eine  |
| 与格 | einem | einem | einer |
| 属格 | eines | eines | einer |

## 否定冠词

否定冠词是德语独有的。

|      | 阳性   | 中性   | 阴性   | 复数   |
| ---- | ------ | ------ | ------ | ------ |
| 主格 | kein   | kein   | keine  | keine  |
| 宾格 | keinen | kein   | keine  | keine  |
| 与格 | keinem | keinem | keiner | keinen |
| 属格 | keines | keines | keiner | keiner |

# 代词

## 三种语序

德语句子的语序分为三种：正语序、反语序和尾语序。这三种语序的区别在于谓语动词的位置。
这三种语序中，谓语动词分别位于句子的第二个位置、第一个位置和最后一个位置。
这三种语序的用处见下表。

| 语序   | 用处               |
| ------ | ------------------ |
| 正语序 | 陈述句、特殊疑问句 |
| 反语序 | 一般疑问句、祈使句 |
| 尾语序 | 从句               |

可以看出，英语中特殊疑问句和一般疑问句的语序和德语是一样的。

# 动词

德语的动词有很多种变形，这里简单起见，分为以下几种：不定式、一般现在时、命令时、第二分词、一般过去时、第二虚拟式。
这些是需要记忆的最少变形。第一分词和第一虚拟式因为比较少用，这里就不介绍了。

## 不定式

在德语词典里查到的单词都是不定式，可以认为不定式是德语单词的“原型”。不定式都以-en结尾。
